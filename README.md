# germanenguix.github.io

Sitio web personal en construcción.

## Stack Tecnológico

- **Framework**: Next.js 15 (App Router)
- **UI Library**: React 19
- **Styling**: Tailwind CSS v4
- **UI Components**: Shadcn UI
- **Language**: TypeScript

## Desarrollo

```bash
# Instalar dependencias
npm install

# Modo desarrollo
npm run dev

# Build para producción
npm run build
```

## Deploy

Este proyecto está configurado para GitHub Pages con exportación estática.
